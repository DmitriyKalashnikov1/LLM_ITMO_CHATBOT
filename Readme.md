## Что это?
Это образовательный бот на основе больших языковых моделей (LLM), задача которого помогать студентам лучше усваивать курс по ИИ от Университето ИТМО.

## Как это запустить?
Для запуска данного бота в ОС Ubuntu Linux нужно:

1) Скачать с офф.сайта программу LLM Studio
2) Запустить в папке бота настройку системы, передав скрипту настройки путь до скаченного файла .appimage (pathToAppImage)
```bash
python -m venv .venv
source .venv/bin/activate
./setupLMStudio pathToAppImage
```
3) После успешного завершения настройки запустить бота нужно командой
```bash
./startLMStudio
```
